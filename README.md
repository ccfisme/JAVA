继承我理解为就是一个类完全复制另一个类的结构，包括函数，变量，可以改变复制的原生类的结构，可以拥有自己的属性和方法，即子类可以对父类进行扩展，子类可以用自己的方式实现父类的方法  


出现原因：代码臃肿，对单个子类修改时不好改
